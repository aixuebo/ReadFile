一、删除
1.删除某一行
sed '2d' file > outFile
表示删除第2行数据,然后重定向到新文件里面

2.删除第1行到第三行内容,即删除前三行内容
sed '1,3d' file > outFile
注意[1,3]是闭区间


3.$d表示最后一行
删除文件的第2行到末尾所有行：
sed '2,$d' file

删除文件最后一行：
sed '$d' file

4.删除文件中所有开头是test的行：
sed '/^test/'d file


5.删除空白行：
sed '/^$/d' file


二、替换
将双引号取消
sed 's/"//g' file
语法是 sed 's/原始内容/替换后的内容/g'



demo
cat file | sed 's/"//g'  | sed '1d' >out.log
表示读取文件,然后将双引号取消,然后删除第一行数据,将结果输出重定向到out.log中
