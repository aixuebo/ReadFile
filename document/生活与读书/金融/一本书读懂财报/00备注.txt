一、其实就是简单的维度计算的过程
现金表、利润表、其实相当于在做group by操作。具体的维度已经明确，需要把一段周期内的数据按照维度做sum汇总。
资产负债表，相当于自己银行卡存款，每次看一下当前有多少钱，表示一个快照。

二、投资
资产负债表、现金表都有投资模块，投资不代表一定投资到外面企业，投资自己也算投资，比如新建厂房，投资到新的地区拓展业务。投资到新的领域。

三、企业活动的流转
融资--先拿到钱。
投资 ---构建厂房 。
经营----购买原材料，生产，销售，获取现金。

四、如何通过三个报表识别一个好的公司
1.资产负债表
查看资产情况是否值钱，不虚。
比如华谊兄弟，股权都被质押了，同时资产都是明星子公司，不产出价值，但资产价格又极贵，资产价值较虚，不行。
工厂固定资产等无形资产评估难度大，因此资产是否值钱还是看现金资产是否多更好。
应收款要尽量小。

应收账款、存货等资产是有风险的，比如存货会随着市场价格波动有变化，应收账款可能要不回来，因此要注意及时的在利润表中做资产减值。

2.现金流量表
经营现金流要为正，说明企业造血能力没有问题。
融资要少，企业优先使用自有资金投资发展、其次使用债券投资发展。最后才会股权。核心是好的企业对自己有信心，不会用股权的方式让其他人获取收益。
投资可以有，扩大生产。投资的前提是自由资金充足。

3.利润表
营业利润是否充足。
充分过度关注资产减值部分，这说明企业就不是好企业的标志。
所得税是否在25%以内。
营业外利润占比小。并且没有政府补贴

五、三个报表之间的关系
1.现金流量表，可以反映公司的经营风险以及公司处于哪个投资阶段，参考08节。  说明公司有钱
2.资产负债+利润表，反映公司的运营状态。  说明公司会赚钱。
3.资产负债表 记录的支出体现在未来对公司有价值
利润表记记录的支出体现在对当前企业有价值。
4.利润表增加，可能是销量好，也可能是涨价造成的。因此要对产品价格相对熟悉才好判断原因。销售好才是真的好。如果涨价造成的，同时存货也增加了，那可能是风险，说明利润的增加的代价是货销售不出去了。
5.资产负债表中存货的增加，可能是物价上涨造成的存货增加(相对安全，上游涨价了，控制不了)，还是卖不出去造成的存货积压（这个很重要，说明企业在衰退）
6.利润表正向利润 - 现金表正向流入 差值，就是资产负债表资产的增量，查看增量在哪些项目上，是否有风险，比如增加到应收账款里就有风险。
即赚到的钱，转换成现金 或者 资产负债表中资产。这是恒等式。

六、核心比率财务指标公式
1.评估公司的运营能力
a.净利率 = 净利润 / 营业收入
b.毛利率 = 毛利润 / 营业收入
c.总资产
总资产周转率 = 营业收入 / 平均总资产 = 营业收入 / （(年初总资产 + 年末总资产) / 2），即给定总资产情况下，能带来多少收入，比如资产100万，能带来1000万收入，说明循环了10次。
总资产周转天数 = 365 / 总资产周转率，即一年365天，资产被循环了N次，即计算每次天数。
以此类推：总资产替代固定资产、流动资产，就会有“流动资产周转率”、“流动资产周转天数”、“固定资产周转率”、“固定资产周转天数”

d.存货 -- 只是转换成营业成本，因此分子不是营业收入
存货周转率 = 营业成本 / 平均存货 = 营业成本 / （(年初存货 + 年末存货) / 2）
存货周转天数 = 365 / 存货周转率

e.应收账款
应收账款周转率 = 营业收入 / 平均应收账款 = 营业收入 / （(年初应收账款 + 年末应收账款) / 2）
存应收账款周转天数 = 365 / 应收账款周转率

f.ROA = 总资产回报率 = 净利润 / 平均总资产 = 净利润 /（(年初总资产 + 年末总资产) / 2）
g.ROE = 净资产利润率 = 净利润 / (资产 - 负债) = 净利润 / 股东权益。
