一、概念
1.同型分析 --- 趋势分析、比较分析。
2.比率分析企业盈利能力。

二、核心内容
1.现金表，关注企业风险。揭示企业的风险 与 持续经营能力。
资产负债表和利润表，是关注企业能活下来的情况下，有多少家底、多少利润，即收益视角看公司。
2.核心比率财务指标公式 --- 评估公司的运营能力
a.净利率 = 净利润 / 营业收入
b.毛利率 = 毛利润 / 营业收入
c.总资产
总资产周转率 = 营业收入 / 平均总资产 = 营业收入 / （(年初总资产 + 年末总资产) / 2），即给定总资产情况下，能带来多少收入，比如资产100万，能带来1000万收入，说明循环了10次。
总资产周转天数 = 365 / 总资产周转率，即一年365天，资产被循环了N次，即计算每次天数。
以此类推：总资产替代固定资产、流动资产，就会有“流动资产周转率”、“流动资产周转天数”、“固定资产周转率”、“固定资产周转天数”

d.存货 -- 只是转换成营业成本，因此分子不是营业收入
存货周转率 = 营业成本 / 平均存货 = 营业成本 / （(年初存货 + 年末存货) / 2）
存货周转天数 = 365 / 存货周转率

e.应收账款
应收账款周转率 = 营业收入 / 平均应收账款 = 营业收入 / （(年初应收账款 + 年末应收账款) / 2）
存应收账款周转天数 = 365 / 应收账款周转率

f.ROA = 总资产回报率 = 净利润 / 平均总资产 = 净利润 /（(年初总资产 + 年末总资产) / 2）
g.ROE = 净资产利润率 = 净利润 / (资产 - 负债) = 净利润 / 股东权益。

三、银行、投资人、并购更偏向看企业哪个报表
1.银行更愿意看 现金表。
银行关注现金流水，贷款给企业的钱可以更有保障的回收回来。
2.投资人更关注利润表
投资人肯定假设前提是 投资一个风险小的企业，要的更多的是企业利润，所以更关注利润表，公司到底赚不赚钱是投资的重点。

关心持续利润，而不是关心非持续利润（营业外收入、政府补贴收入、汇兑损益）

3.企业并购更关注资产负债表。
一旦企业被收购了，经营内容都可能发生变化，因此看利润表也没有意义了。
他更想看明白 企业有什么东西，买的是公司的 资产-负债，即并购买的是所有者权益。即既买了企业的资产，也买了企业的负债。

四、收购时，资产和负债都可能存在风险。
1.财务报表被粉饰后
资产很多，会产生被高估资产的风险。
负债少，会低估风险。
2.评估公司只能评估公司的资产，但不能排清楚公司负债。
因为评估公司也不知道 没有被记录的负债，比如或有负债(为他人担保的负债)。

五、同型分析
1.将利润表、现金表、资产负债表中每一项占比从大到小计算好，并且排列。更清晰的了解企业的情况。
比一般财务报表更直观。

利润表，可以知道如何一步步从收入到利润的。
资产负债表，可以知道资产构成 与 资金来源。

2.比如 -- 利润表
营业成本占收入72%。营业费用占比6%、管理费用占比5%、财务费用占比3%、毛利润占比45%、营业利润占比14%、净利润占比12%。

3.比较
比较分析：和同行业比较。
趋势分析：和自己过去比较，了解每一项的趋势变化。


六、比率分析企业盈利能力
1.基于企业的财务数据，进行加减乘除运算，产生计算指标。通过计算指标分析企业。
比如老板看到毛利润是1200万，净利润是520万，就会懵，自己公司到底是赚的多还是少呀。没概念。

因此转换成比例就更容易了解。
毛利率 = （收入 - 支出）/ 收入 = 28%。
净利润率 = 税后净利润 / 收入 = 12%。反应销售后公司能赚多少钱的比例。假设净利润率不变的情况下，收入越高，那肯定赚的越多呀。

即净利润率高，说明企业从收入到利润过程中损耗较少，企业是盈利能力不错的企业。因此利润率是与效益有关的概念。

2.总资产周转率
企业进原材料--生产-- 卖出--获取现金 -- 购买原材料 --进入循环。
一年循环多少次，就是总资产周转率的概念。

总资产周转率 越大，完成循环的次数越多，赚钱越多。 因此总资产周转率是一个与效率有关的概念。
即 收入/平均总资产，即 总资产一定的情况下，收入越高，说明周转率越高。

3.如何评估效率和效益的综合指标 ---- 总资产周转率（效率）、净利润率（效益）。
如果效率和效益都高，才是更好的，任何一个高，另外一个低，都可能不是好事儿。
因此企业的效率和效益都要分别关注。

ROA = 总资产回报率 = 总资产利润率 = 资产总额利润率 = 总资产报酬率，评估企业运用全部资产的总体收益能力。
ROA = 总资产周转率 * 净利润率 =（收入/平均总资产） * （净利润/收入） = 净利润/平均总资产。即总资产一定的情况下，产生利润越高，什么ROA回报率越高，越是好事儿。
因此ROA越高越说明利润越高，越好。


ROE = 净资产利润率 = 净利润 / (资产 - 负债) = 净利润 / 股东权益。
数据越高，说明给股东带来的收益越高。


七、评估公司短期债务偿还能力的指标
短期债务指代一年内要还的债务，包括付给供应商的钱，员工工资等。

1.流动比率 = 流动资产 / 流动负债。比例越高，说明短期偿还债务能力越强。
因为短期债务，企业可以通过现金 + 应收账款 + 存货等快速变现，去偿还。
显然流动比率过于理想，因为存货要卖了才可以转换成应收账款，还要继续转换才能到现金。

2.升级版 ： 速动比率 = (流动资产 - 存货) / 流动负债.

3.计算后的比例，多少算好呢？
比如结果是1，是好么？相当于流动资产刚好还债，企业也没有流动资金用于正常运营了。

因此流动资产扮演了两个角色，一个是企业偿还短期债务提供资金，另一个是公司日常运营提供资金。

经过分析国外上市公司历史情况，发现在破产前，大多数企业的流动比例是2，即不要以为流动资产>2倍流动负债就是安全的，恰恰是快倒闭的特征。
正常情况应该是3~4之间。

但中国公司大多数企业流动比率只在1~2之间，这是为什么呢?是因为中国有特色借新债还旧寨的方式，而不需要企业靠流动资产变现偿还短期债务。
因此除非行业出现问题，银行突然停止贷款，否则他就能好好活着。

中国企业应该看的是 流动资产 / 短期借款流动负债，这个比例就不低了，大概在4~5之间，因此中国企业应该靠这个指标来观察，

八、评估公司长期债务偿还能力的指标
1.能不能用 长期资产/长期负债来评估呢？
不行，是因为短期资产是可以卖掉，是正常的企业操作，但长期资产是什么，是厂房设备，他不可能卖掉，如果卖掉企业就不经营了。所以这个指标不能评估。

2.如何评估
两个维度，偿还利息、偿还本金。
a.利息 通过企业利息税前收益评估。因为净收益是扣税 并且扣除利息的。
所以 利息税前收益 = 净利润 + 利息 + 利息扣税。 
利息税前收益  / 利息 = N倍，N越大，说明企业偿还利息的能力越强。（这个利息税前收益还是比较复杂计算的，如果利息不多的情况下，或者净利润/利息后N很大的情况下，就不用这个复杂的计算利息税前收益了）。

b.偿还本金
资产负债率 评估 长期还本金能力，当比例很大的时候，也说明企业借钱太多了，风险大。
主要是偿还本金的指标太不好评估，所以暂时拿资产负债率评估也是可以接受的。
