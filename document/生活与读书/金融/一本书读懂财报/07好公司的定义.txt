收入规模、利润水平看，公司成长性都是非常高的。显然就是值得投的好公司，赚钱的公司。

2.我们看一下他的投资回报率
a.总资产回报率 = 净利润 / 总资产。描述企业债权人和股东所有的综合投资回报率。或者说是企业整体的投资回报率。即总资产 = 负债 + 股东权益。
b.净资产回报率 = 净利润 / 股东权益。描述的是股东的投资回报率。

c.投资资本回报率 = 税后营业利润 / 可投资的资本
因为a和b计算上有一些问题，所以才有了c。
参考利润表，净利润是扣除所得税，扣除财务费用(债务利息)。
总资产 = 债务 + 股东权益 + 欠供应商的钱 + 欠员工的工资 + 欠税务局的税收。

因此 a中 分母总资产中，欠供应商的钱 + 欠员工的工资 + 欠税务局的税收 这三个部分不是可投资的汇报，因此不应该加进来算回报率。
b中，股东权益虽然是可以计算投资回报率，但不是企业利用杠杆的全部，因为债权也是杠杆的一部分，因为企业为这部分杠杆支付了利息。

所以 
可投资的资本 是 总资产的子集 = 股东权益 + 债权。
税后营业利润 是股东权益 + 债权 获取的利润 ，因为净利润 是 利润 - 债务利息 - 所得税。所以税后营业利润 = 净利润 + 债务利息即可。

四、投资资本回报率多少算好呢？
1.肯定越高越好，说明回报率高，资金效率高。
但及格线是多少呢？是银行利息 还是 通货膨胀率？

2.其实及格线应该是企业使用投资人的资本成本。
即 回报率 - 投资人钱的资本成本 > 0,投资人资本赚到钱了。是超额收益率，是理想的。
问题是 “投资人钱的资本成本”很难评估。

先说一下结论，该公司经过分析，发现只有第一年是有超额收益率的，后两年其实是亏钱的。所以投资这个公司是要值得反思的。

五、如何计算 “投资人钱的资本成本”
1.债权的成本 取决于银行利率，这个是标准的，所以成本好计算。
2.股东的成本，不好计算，相当于机会成本，我们可以拿同行业的均值算股东的成本。
因此加权平均成本计算即可，即经济利润 = 债权金额 * 债权成本 * （1-所得税税率） + 股权金额 * 股权成本。

注意:
比如所得税是25%，则1 - 0.25 = 0.75.
为什么要*0.75呢，是因为净利润 = 利润 - 债权利息 - 所得税。
如果两个公司净利润相同的情况下，一个公司债权多，就会利息多，所以扣税相对少。所得税 = 扣利息后*25%。


当然忽略 * （1-所得税税率）也是可以简化公式的。

六、什么公司是赚钱的好公司
净利润>0，说明公司赚到钱了。经济利润也要>0，说明股东的投资资金成本>行业均值了。
如果净利润>0，经济利润<0，说明公司赚到钱了，但没有考虑股东的资金成本，如果算上股东资金成本，是亏钱的，不如股东投到行业均值里赚得多呢。那股东投资你的意义是什么？
