一、背景与总结
背景知识参考 汇率结算.txt

1.开放经济的好处
贸易使人们生产自己最擅长的东西,消费其他国家生产的各种物品。(换成个体也一样算生活在开放经济中)

2.公式
储蓄 = 国内投资 + 国外投资 = 国内投资 + 资本净流出(负数/正数)
资本净流出 = 储蓄 - 国内投资。

3.资本净流出 = 资本流出 - 资本流入。即 本国居民购买国外资产 - 国外居民购买本国资产。
资本净流出为负数，说明该过与资本流入。

二、基础认知
1.基本概念
a.进口、出口。
b.贸易余额 = 净出口 = 出口 - 进口。
c.贸易盈余、贸易赤字、贸易平衡，分别代表净出口的正数、负数、0。

2.影响净出口的因素
a.消费者对国外物品的嗜好程度。
b.国内外商品价格差异。
c.汇率。
d.国内/国外消费者的收入。
e.运输成本。
f.政府的国际贸易政策。

3.案例研究
a.可以参考书中图，(进口+出口)/GDP的比重,1950年前大概5%，2010年已经12%；1980年前进出口相对平衡，但1980后，贸易赤字明显。
b.主要贸易增加的原因：
运输的改善，1950年商船运载量不足1万吨，现在超过10万吨；
空运成本也更加便宜；
通信技术的进步，企业之间可以更容易的对接国外客户。
随着时间的推移，各国间接受了自由贸易是互利的理念，因此政府政策的改善也是重要原因，在降低关税方面也在做功。
c.体积大的原材料(钢铁)、容易腐蚀的物品(食物)比重偏小，但体积小、重量轻、容易运输的家电、粮食、电影业是美国的主要出口品。

4.什么是出口
a.林书豪算出口吗？算，带动了NBA畅销商品、球迷的旅游、球迷买球票。
b.外国学生支付学费、支付飞机票、消费谷歌上外国广告主的广告。
c.文化上改善外国人对美国的看法，让更多的人来参观美国、把孩子送美国读书，因此放宽学生和旅游签证，也是刺激出口的方式。
比如鼓励降低航空税、降低签证费用、降低机场降落费用等。减少旅游、留学的签证手续，增加进美国的旅游人口、留学人口，这些人都会在美国产生消费，而且是主观上自己愿意的消费。
以上都是服务业的新出口，美国在新出口上是贸易盈余的，并且逐年在扩大盈余，为美国创造了更多的就业岗位。
新出口不需要与中国进行外交战，也不需要国会投票贸易协定，这些出口可能都是其他国家的需求，想给与美国的机会，但美国短时却与其失之交臂。

三、金融资源的流动
一个美国居民，可以用8万元买一辆国外汽车，也可以用8万元购买国外债券、基金、股票，后者就是资本流动。
1.基础概念
a.资本流入（国外买本国资产）、资本流出(本国买国外资产，不看好本国经济的象征)。
b.资本净流出 = 资本流出 - 资本流入。即 本国居民购买国外资产 - 国外居民购买本国资产。
资本净流出为负数，说明该过与资本流入。

2.资本流动的形式
a.投资建厂。比如去国外开一家麦当劳。
b.购买股票、债券。

3.影响资本净流出的因素
a.国外资产的真实利率、国内资产的真实利率。
b.经济、政治风险。
c.影响国外对国内资产所有权的政府政策。
比如买美国还是墨西哥债券，在利率一样时，肯定购买美国更稳妥。

4.净出口 = 资本净流出
原因是每一笔交易影响左边，也必定影响右边，所以是恒等式。
a.举例
美国程序员，写了软件，1万日元卖给了日本。增加了净出口，收入是1万日元。
那1万日元如何影响资本净流出的呢？
①放家里，说明得到了外国通货资产,相当于投资日本经济，即期望日元未来会升值。资本流出。
②直接投资日本经济，买日本公司股票/债券。结果也是拥有了国外资产，资本流出。
③直接购买用日元，购买日本的商品，结果属于进口商品，净出口 = 0。
④直接换成美元，用美元购买美国的进口日本商品，结果属于进口商品，净出口 = 0。
⑤直接换美元，但银行会把1万日元也会进行购买日本资产(股票、债券、持有日本通货)的资本净流出行为；也可以用日元购买日本物品(进口),也可以兑换给其他需要日元的美国人(美国人依然去做投资或者购物)。
b.举例
简单来说，中国看做一个整体，贸易盈余，赚了5000万美元，可以持有做货币储备、购买美国国债/股票(资本流出)、购买美国大飞机(进口)、兑换给国内需要美元的人们。
c.结论
一国的贸易盈余，一定去购买其他国家的资产(货币储备(钱也是一种保值资产)、资本流出、进口)
一国的贸易赤字，一定需要去出售其他国家的资产。好比家庭都没有钱了，凭啥会赤字，一定是把持有别人家的东西卖了才能换自己需要的东西。特例美国除外，他可以印钱，美元相对其他国家来说，都算其他国家的资产。

四、储蓄、投资、
1.公式
GDP = 消费 + 政府消费 + 国内投资+净出口 ==>  GDP - 消费 - 政府消费 = 国内投资 + 净出口 ---> 储蓄 = 国内投资 + 净出口 ---> 净出口 = 资本净流出 ---> 储蓄 = 国内投资 + 资本净流出 --> 储蓄 = 国内投资 + 国外投资。
即赚到的储蓄，一定是要么投资国内、要么投资国外。
2.总结
储蓄 > 国内投资时，说明资本净流出，去国外投资了。
储蓄 < 国内投资时，说明资本净流入，其他国家资金投入到国内，国内发展被外界看好，即资本净流出是负数，表示外国人在购买国内资产比重很大。

3.案例 -- 美国贸易赤字的国家
a.美国1960-1980，国内储蓄 与 国内投资，基本相等，即资本净流出接近于0.
1980年后，国内储蓄 << 国内投资,引起相当大的贸易赤字。即国内投资的很多钱是来自于 资本流入。(赤字原因，别人手里有很多美元，只能投资到美国)，贸易赤字越大，资本流入越多，国内投资越比国内储蓄大的gap非常大。
b.资本净流出 = 资本净流出 = 资本流出 - 资本流入，赤字越大，gap越大，并且结果一定是负数。
c.通常用资本流入的金额/GDP金额，这个指标做为抓手。
相当于投资美国的钱占比GDP越多，相当于这部分钱回报率越大，不管是股权投资还是债权投资，都要最后借款人要拿到收益的，假设极端情况，指标100%，说明美国经济都依赖于资本流入，别的国家赚的非常多，美国经济未来一定不是好的经济。还款压力太大了。
这就是赤字带来的代价。毕竟一直在借钱过日子，总要还的。
d.赤字的原因 = 资本净流出的原因
①不平衡的财政，1980-1987，政府赤字增加(减税政策 + 增加国防支出)，减少了国家公共储蓄，从而减少了国内储蓄。缺钱，但还要正常花钱，别人就赚到了美元，反过来别人可以借你钱，所以资本流入多。
②投资高涨，1991-2000，这期间，政府赤字为盈余，国内储蓄是增加的，但国内发展非常繁荣，许多高科技行业需要钱投资，未来回报率高，因此吸引了国外资本的流入。
这个阶段赤字是好事儿，因为投资产生了效率，生产更多物品，那么就会有好的收益，赚钱可以处理咯是欠下的债务。缺点也明显，如果投资没收益，相当于债务更严重了。
③经济衰退，2000-2012，国内储蓄、投资都下降了，因为2008年后经济很难，投资收益非常小，所以大家不愿意投资。储蓄减少也是经济不好做出的反应，没有钱储蓄了，政府也赤字没钱消费。
好在美国钱全球硬通货，全球把便宜物资卖给美国，换美元，反过来投给美国，造成的赤字。(换其他国家，这么玩都废了)

4.贸易赤字是不是好事儿
看情况，贸易赤字相当于欠别人钱去发展自己，如果只用于吃穿的挥霍，肯定不是好事儿，但发展高科技，如果成功了，那赤字是好事儿，借别人的储蓄，发展自己，反过来拿发展的结果去卖给别人。(比如美国大飞机、美国武器等)。

五、汇率
1.定义
a.名义汇率 -- 衡量的是货币
规定好一个国家的通货交换另外一个国家通货的比例，比如1美元=6人民币。
一个国家通货与不同国家通货比例是不同的，所以会存在N个名义汇率。而大家说的汇率升值/贬值，不是指代某一个名义汇率，而是指代一揽子名义汇率的指数。
b.真实汇率 -- 衡量的是物品
一国物品交换另一国物品的比例。比如中国的手机价格是美国的3倍，在手机上，真实汇率就是1:3。
因此也是对应非常多的商品，就非常多的真实汇率，平时研究真实汇率，也是一揽子商品的真实汇率指数。

2.汇率升值(坚挺，可以买更多国外通货)/贬值(疲软)
升值与贬值是相对的。
如果1美元可以买更多国外通货，说明美元升值，对立方贬值。
相反买更少的国外通货，说明美元贬值，对立方升值。

3.欧元 --- 为什么欧洲要统一货币
a.贸易更容易，如果中国每一个城市都不同的货币换算，那肯定没办法更容易的交易贸易。
b.政治决策，跟秦始皇统一语言类似，想削弱民族主义感情，使欧洲人更充分意识到大家的共同命运与历史。认为货币统一有助于这个发展。

4.真实汇率与名义汇率的关系
a.背景
美国大米是100美元，日本大米卖16000日元，而名义汇率是1美元 = 80日元。
转换同一单位 ，美国大米 = 100*80 = 8000日元。
b.真实汇率是1:2。
c.公式推导
真实汇率 = (名义汇率 * 国内价格)/国外价格。
= (名义汇率e * 国内一揽子国币指数p) / 国外一揽子货币指数p*
= (e * p) / p*

d.结论
真实汇率 取决于 名义汇率 与 本国通货。因此名义汇率会影响真实汇率，从而决定人民到底选择国内商品还是国外商品。

六、第一种汇率决定理论 -- 购买力平价
1.基础假设，一种物品在所有地方都应该相同的价格出售，否则就会被套利。即一种通货必然在所有国家具有相同的购买力，购买力是能购买到的物品衡量的货币价值。
同一个商品，在不同国家价值是一样的，因此影响汇率长期的变化因素。
购买力平价 = 任何一单位通货应该在所有国家买到等量物品的汇率理论。
比如1美元可以买一个美国苹果，6人民币可以买一个中国苹果，那么以苹果视角看，美元与人民币汇率就应该是1:6。即真实汇率决定名义汇率。
这也就解释了为什么一些高速通货膨胀的国家，汇率极度不稳定，1美元可以换627意大利，变成1美元可以换1700意大利。原因就是意大利印钱，造成的通货膨胀影响了物价，从而影响汇率。

2.基于购买力平价理论，真实汇率是稳定的，因此真实汇率是在两个国家是固定值。
真实汇率 = (名义汇率 * 国内价格)/国外价格。 公式改写，将真实汇率设置为1常量，即1 = (名义汇率 * 国内价格)/国外价格 --> 名义汇率 = 国外价格 / 国内价格。
即名义汇率受两国的物价水平影响。
因此一个国家货币量增加，导致物价水平增加，则该国的通货相对于其他国家而言就贬值。
(如果控制住通货膨胀，货币量增加也无所谓，比如中国货币增加在房子的蓄水池里，生活物品没有造成通货膨胀，因此汇率是稳定的)

3.真实汇率 、名义汇率 与 物价水平不是不一定是同步的，比如汉堡包换算的不同国家的真实汇率，与名义汇率结果可能是不同的，但可以基于“购买力平价”推导出一定的名义汇率趋势，用于日常投资维度分析。
