# 背景与总结
1.倾听很重要。有一些话可以不用说，不要什么话都说；交流的时候如果遇到场景，就停止交流；明确复述，让对方知道你理解的是对的。

# 一、学会倾听
1.倾听很重要，因为听的过程可能会溜号，造成听到的内容是拼接而来的，因此就会有没听懂或者说过但忘记了(其实是被选择性忽略了)。
比如甲向乙提”你当初不是说..“，乙很惊讶，我什么时候说过，其实乙可能真没说过，是甲理解成另外一个含义了。

2.为了做的有效倾听，最好的策略是克制"过早质疑”
a.就算要质疑，也要等到对方把话都说完；可能说完后，前期准备质疑的点，就不在是质疑的内容了。
b.不要相信记忆力，要拿着纸币记录好要质疑的内容。

3.反驳或者倾听后语气不好的回复，可能会造成最后的争吵，而不是交流。
所以倾听过程，少说话，是非常重要的。

# 二、说与不说
1.明明同样的话，自己数出来就不像那么回事了。
不是班里学习最好的学生讲学习方法，很多同学都不屑听。

2.知道什么话能说，什么话不能说
根据自己的情况，选择符合自己身份该说的话，说别人能听的话。

3.通过努力，积累成为能说更多话的人
比如马云，他的成功，让他说话别人更爱听。
一定要想办法做出令人敬佩的事情，这样就会主动有人找你做朋友。

因此我们我们有时候说话，身边的人听不进去，我们会觉得失落、愤怒，其实只是我们还没有赢得足够的重视，我们需要继续积累和努力。

4.自我成功评估的方法
判断说出来的话，有没有人愿意听。

# 三、交流守则
1.一个事实：不同的人对相同的事情，看法和观点多少会存在一些差异。
那么交流过程中如何处理差异带来的问题呢？ 有三个原则要坚持。

2.原则1：有意义的讨论的前提是双方不仅要相互竞争，也要相互合作。一旦忽视、违背该原则，则马上停止交流。
比如秀才遇见兵，有理说不清，因为讲道理的是需要前提的，两者不存在这个前提，因此不要去交流，强行交流肯定会带来痛苦。
比如生活中遇到不遵守理性讨论原则的讨论会，应该停止交流。比如聊天内容：本来就这样呀；行行行，你最聪明；就算道理是那样，我也无法接受；你自己看着办吧；不管了；放屁；滚蛋；
上述内容发生时，就要马上停止交流，不要天真的以为可以对这些人讲清楚、解释清楚现状，在把讨论拉回正轨，试图这么做的人，多少都会被证明是徒劳的。
就算对方不依不饶，我们也要回避，回避，回避，因为接下来讨论的过程都已经不在是讨论，而更可能是争论、辩论，这种争论的代价是浪费时间、影响情绪、对方变成敌人、失去更多朋友。

3.原则2：真理独立存在，他不依附于任何个人或者集体。-- 因此遇到自以为是的人，立刻退出讨论。
a.自以为是是每一个人成长过程中必须经历的阶段。
因为每个人生来都是以无知作为起点，而成长之路原本就是求知之路。
交流讨论中，有这类人存在，讨论不会得到有意义的结果，只能在不断地被他说服的过程中，自己是否接受。

4.原则3：真理是不变的，更不会因为人而变，不停变化的只是人们对真理的解释。
如果在交谈中，一方说”每个人都应该有自己的看法，不是么?“,或者"我保留我的看法"，谈话就应该结束了。因为他这么说的潜台词就是对这个事实的理解已经不想改变了。
我们应该讨论事实，而不是看法，看法肯定是每一个人不一样的，没什么好讨论的。我们互相交流看法，各自考虑哪个更有道理，就够了，看法本不应该被讨论。

# 四、正确复述
1.为什么要复述
每当接收到交流的信息后，我们真的能准确理解发送方要表达的真实内容么？其实是很难的，信息是有损耗的，与上下文缺失有关系。
因此为了保障沟通顺利，往往需要添加一个验证机制，或者是反馈机制。

2.好的复述话术
你的意思是...是吗？我理解的对吗？

3.不好的复述话术
a.那你的意思是...喽?
b.然后不等对方反应,就发起攻之。(本质上他们认为自己完全理解对方，从来没有认知到接收的信息可能不是自己理解的，总是认为自己是对的，是聪明的带来的麻烦)
